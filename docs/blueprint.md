# **App Name**: Family Financials

## Core Features:

- Spending Overview: Displays a visual summary of income, expenses, and overall financial status for the family.
- Income Tracking: Allows users to input and track all sources of income.
- Expense Tracking: Enables users to categorize and log all expenses.
- Deficit/Surplus Calculation: Automatically calculates the difference between total income and expenses to show the current financial balance.
- Personalized Financial Advice: Offers tailored financial advice and insights based on spending habits using an AI tool.
- Interactive Charts: Generates charts and graphs to visualize financial data, trends, and spending patterns.

## Style Guidelines:

- Primary color: Forest green (#558B2F) to evoke trust and growth.
- Background color: Light gray (#F0F4F0), providing a neutral backdrop to ensure readability.
- Accent color: Burnt orange (#D84315), highlighting important actions and insights.
- Font pairing: 'Inter' (sans-serif) for body and 'Alegreya' (serif) for headers.
- Use simple, modern icons to represent financial categories and actions.
- Emphasize a clean, spacious design for easy readability and navigation.
- Use subtle transitions to show data updates smoothly, avoiding distractions.
